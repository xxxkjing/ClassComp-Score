# 信息委员使用手册

本手册面向信息委员用户，介绍如何使用系统进行电脑卫生评分。

---

## 📋 角色定位

**信息委员（Info Commissioner）**是系统的基础评分人员，负责按照评分链条对指定年级班级进行定期评分。

### 核心特征

- **评分权重**：1.0x（基础权重）
- **评分范围**：按评分链条对指定年级评分
- **主要职责**：定期评分、记录问题、查看历史

### 评分链条规则

信息委员按照以下链条进行评分：

```
中预班级 → 初一班级 → 初二班级 → 中预班级
         ↑________________________↓

高一班级 ↔ 高二班级

高一VCE班级 ↔ 高二VCE班级
```

**说明**：
- 中预班级的信息委员评初一班级
- 初一班级的信息委员评初二班级
- 初二班级的信息委员评中预班级
- 高一和高二班级互相评分
- 高一VCE和高二VCE班级互相评分

## 🚀 快速开始

### 首次登录

1. 访问系统地址：`http://your-domain.com`
2. 使用信息委员账户登录：
   - 用户名：如 [`中预1班`](中预1班)、[`初一2班`](初一2班)
   - 密码：管理员下发的初始密码

3. **首次登录需要注册**：
   - 输入班级名称
   - 验证初始密码
   - 设置真实姓名
   - 设置新密码

4. 使用新密码重新登录，进入评分首页

**⚠️ 重要**：首次登录必须完成注册流程，设置新密码后才能使用系统！

### 界面概览

登录后的评分界面包含：

- **周期信息卡片**：显示当前周期号、类型、时间范围
- **待评分班级列表**：根据评分链条自动显示需要评分的班级
- **评分表单**：输入各项分数和备注
- **我的评分**：查看个人评分历史记录

## 📝 评分流程

### 第一步：查看待评分班级

1. 登录后自动进入评分首页
2. 系统根据您的班级自动显示需要评分的目标年级
3. 可以看到：
   - 该年级所有需要评分的班级
   - 每个班级的评分状态（已评/待评）
   - 上一次评分的时间

**示例**：
- 如果您是中预1班的信息委员，系统会显示所有初一班级
- 如果您是高一3班的信息委员，系统会显示所有高二班级

### 第二步：填写评分内容

点击"开始评分"后，填写评分表单：

#### 评分项目

**1. 电脑整洁度（0-3分）**
- 键盘、鼠标、显示屏是否干净
- 没有灰尘、污渍、指纹
- 接线整齐，没有缠绕

**2. 物品摆放（0-3分）**
- 电脑周围物品摆放是否整齐
- 桌面无杂物堆放
- 垃圾桶及时清理

**3. 使用规范（0-4分）**
- 椅子是否归位、摆放整齐
- 没有损坏设备的情况
- 遵守机房使用规范

**每个班级总分 = 整洁度 + 物品摆放 + 使用规范 = 满分10分**

#### 评分标准参考

| 分数范围 | 等级 | 标准说明 |
|---------|------|---------|
| 9-10分 | 优秀 | 各项都做得很好，无明显问题 |
| 7-8分 | 良好 | 总体较好，有个别小问题 |
| 5-6分 | 一般 | 有较多问题需要改进 |
| 0-4分 | 需改进 | 问题较多，需要立即整改 |

### 第三步：添加评分备注

在每个班级的"备注"文本框中记录发现的具体问题：

**好的备注示例**：
- ✅ "键盘有明显灰尘，第3排显示屏有指纹"
- ✅ "椅子未归位，垃圾桶已满需清理"
- ✅ "网线整理较乱，建议使用理线器"

**不好的备注示例**：
- ❌ "不干净"（太模糊）
- ❌ "还行"（没有具体信息）
- ❌ 空着不填（无法追溯问题）

**备注的作用**：
1. 帮助被评班级了解具体问题
2. 为下次评分提供对比参考
3. 管理员可以根据备注发现系统性问题

### 第四步：提交评分

1. 确认所有班级的分数都已填写
2. 检查备注是否清楚准确
3. 点击 **"提交评分"** 按钮
4. 看到成功提示后，评分完成

**提交后的说明**：
- ✅ 评分立即生效，计入系统统计
- ✅ 可以在"我的评分"中查看
- ⚠️ 同一周期内可以重新提交，新评分会覆盖旧评分
- ⚠️ 被覆盖的评分会保存到历史记录中

## ⏰ 评分周期说明

### 什么是评分周期？

**评分周期**是管理员设置的评分时间段，用于规范评分频率和数据统计。

**两种周期类型**：
- **双周期（14天）**：每14天为一个周期，适合常规学期
- **单周期（7天）**：每7天为一个周期，适合需要密集评分的时期

**周期号**：从0开始计数，如第0周期、第1周期、第2周期...

### 查看当前周期信息

在评分首页顶部的"当前周期"卡片可以看到：

- **周期号**：如"第3周期"
- **周期类型**：单周或双周
- **时间范围**：2024-01-01 至 2024-01-14
- **剩余天数**：还有5天结束

### 周期与评分的关系

**同一周期内的评分规则**：

```
第2周期（1月1日 - 1月14日）
├─ 1月3日：首次评分，提交85分
├─ 1月8日：发现有误，重新评分提交88分
│           → 85分被覆盖，保存到历史记录
└─ 最终记录：88分（作为第2周期的正式成绩）

第3周期（1月15日 - 1月28日）
└─ 可以重新对同一班级评分（新周期，新评分）
```

**要点说明**：
- ✅ 同一周期内，对同一班级可以多次提交评分
- ✅ 新提交的评分会自动覆盖该周期内的旧评分
- ✅ 被覆盖的评分会完整保存到历史记录表中
- ✅ 新周期开始后，可以重新评分（不会覆盖上一周期的记录）

## 🔐 账户安全

### 修改密码

**操作步骤**：
1. 点击右上角用户名或头像
2. 选择 **"修改密码"**
3. 输入当前密码
4. 输入新密码并确认
5. 点击 **"确认修改"**

**密码要求**：
- 至少8位字符
- 建议包含字母、数字、符号的组合
- 不要使用过于简单的密码（如123456、生日等）

### 安全建议

**定期维护**：
- 🔒 每学期至少修改一次密码
- 🔒 不要使用与其他网站相同的密码
- 🔒 不要将密码告诉他人

**使用习惯**：
- ✅ 使用完毕后记得退出登录
- ✅ 不要在公共电脑上保存密码
- ✅ 发现账户异常立即联系管理员

### 遗忘密码怎么办？

1. 点击登录页面的 **"忘记密码"** 链接
2. 输入您的用户名（班级名称）
3. 联系管理员重置密码
4. 管理员会为您设置新的临时密码
5. 使用临时密码登录后立即修改

## 💡 评分技巧

### 提高评分效率

**计划安排**：
1. 了解评分周期，合理安排评分时间
2. 避免在周期最后一天才评分
3. 可以利用课间时间分批评分

**现场评分**：
1. 携带手机到机房现场
2. 逐个教室进行检查
3. 边检查边在手机上记录
4. 当场完成评分提交

**批量处理**：
1. 一次性完成所有班级的检查
2. 集中填写评分表单
3. 一次性提交所有评分

### 准确记录问题

**具体描述原则**：

❌ **不好的备注**：
- "有点脏"
- "不太好"
- "需要整理"

✅ **好的备注**：
- "第1排第3台键盘有灰尘，显示屏有手印"
- "垃圾桶已满，地面有纸屑"
- "椅子未归位，有5把椅子在教室后方"

**位置标注**：
- 说明具体的位置信息（哪一排、哪一台）
- 便于被评班级快速找到问题所在
- 有助于验证是否改进

**建议改进**：
- 在备注中可以提出改进建议
- 如"建议每天放学后清理键盘"
- 如"建议设置专人负责椅子归位"

### 公平公正评分

**统一标准**：
- 对所有班级使用相同的评分标准
- 不因私人关系而有偏袒
- 客观评价，实事求是

**记录依据**：
- 评分依据真实的检查结果
- 备注中记录具体的发现
- 可以作为评分依据的证明

**及时反馈**：
- 评分后可以告知被评班级
- 指出需要改进的具体问题
- 对做得好的方面也要肯定

## 🔍 常见问题

### Q1: 同一个班级在一个周期内能评分多次吗？

**A:** 可以。系统允许在同一周期内多次提交评分，但新的评分会覆盖之前的评分。被覆盖的评分会保存到历史记录中，不会丢失。

**使用场景**：
- 发现之前评分有误，重新评分纠正
- 第一次评分后班级进行了整改，重新检查后更新评分

### Q2: 评分提交后发现有错误怎么办？

**A:** 
- **同一周期内**：可以直接重新提交评分，新评分会覆盖旧评分
- **周期已结束**：无法修改，请联系管理员说明情况

### Q3: 为什么我看不到某些班级？

**A:** 信息委员只能看到评分链条中对应的目标年级班级。例如：
- 中预班级的信息委员只能看到初一班级
- 高一班级的信息委员只能看到高二班级
- 如果看不到应该看到的班级，可能是管理员未配置该班级，请联系管理员

### Q4: 评分分数是如何计算的？

**A:** 
- 单个班级总分 = 整洁度（0-3）+ 物品摆放（0-3）+ 使用规范（0-4）= 10分
- 最终统计时，会计算该班级在所有评分中的加权平均分
- 信息委员的评分权重为1.0x（基础权重）

### Q5: 可以查看其他信息委员的评分吗？

**A:** 不可以。出于隐私保护，您只能查看自己提交的评分记录。如需查看汇总数据，可联系管理员。

### Q6: 忘记在周期内提交评分怎么办？

**A:** 
- 周期结束后无法补交该周期的评分
- 建议在周期内尽早完成评分，避免遗忘
- 可以设置提醒，在周期结束前1-2天完成评分
- 如有特殊情况，可联系管理员说明

### Q7: 评分时网络断开怎么办？

**A:**
- 评分数据在点击"提交评分"前不会保存
- 建议在网络稳定的环境下评分
- 手机评分时注意网络连接状态
- 如果提交失败，重新填写并提交即可

### Q8: 如何知道被评班级是否改进了？

**A:**
- 在下一个周期对该班级评分时，可以对比之前的备注
- 查看"我的评分"历史记录，对比不同周期的评分和备注
- 观察评分趋势，看分数是否提高

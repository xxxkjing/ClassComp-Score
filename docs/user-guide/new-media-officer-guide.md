# 新媒体委员使用手册

本手册面向新媒体委员用户，介绍如何使用系统的特殊权限进行跨年级评分。

---

## 📋 角色定位

**新媒体委员（New Media Officer）**是系统的特殊评分人员，拥有更高的评分权重和更广的评分范围，适合进行专项检查和毕业班评估。

### 核心特征

- **评分权重**：1.5x（是信息委员的1.5倍）
- **评分范围**：可以对所有年级进行评分（不受评分链条限制）
- **数据标记**：评分记录标记为"新媒体委员"来源，便于追溯
- **主要职责**：专项检查、毕业班评估、跨年级评分

### 与信息委员的区别

| 特性 | 信息委员 | 新媒体委员 |
|-----|---------|----------|
| 评分权重 | 1.0x | 1.5x |
| 评分范围 | 按评分链条限制 | 所有年级 |
| 数据来源标记 | "信息委员" | "新媒体委员" |
| 适用场景 | 日常评分 | 专项检查 |

## 📝 评分流程

### 第一步：选择要评分的年级

1. 登录后进入新媒体委员评分页面
2. 在"选择年级"区域，点击要评分的年级按钮
3. 系统会显示该年级的所有参与班级

**年级分类**：

**初中部**：
- 中预（预备年级）
- 初一
- 初二
- 初三（毕业班 🎓）

**高中部**：
- 高一
- 高二
- 高三（毕业班 🎓）
- 高一VCE
- 高二VCE
- 高三VCE（毕业班 🎓）

**毕业班标识**：
- 毕业班级会用 🎓 标记
- 选择毕业班时会显示特别提示
- 提醒关注信息素养和规范使用情况

### 第二步：批量填写评分

选择年级后，系统会显示该年级所有班级的评分表格：

#### 评分项目

**1. 电脑整洁度（0-3分）**
- 键盘、鼠标、显示屏清洁程度
- 无灰尘、污渍、指纹
- 线材整理整齐

**2. 物品摆放（0-3分）**
- 桌面物品摆放整齐
- 无杂物堆放
- 垃圾桶清理及时

**3. 使用规范（0-4分）**
- 椅子归位、摆放整齐
- 设备使用规范
- 无损坏情况

**每个班级总分 = 整洁度 + 物品摆放 + 使用规范 = 满分10分**

#### 批量评分表格

评分表格采用表格形式，一次性显示所有班级：

| 班级 | 整洁(0-3) | 摆放(0-3) | 使用(0-4) | 备注 |
|-----|----------|----------|----------|------|
| 中预1班 | [3] | [3] | [4] | 优秀 |
| 中预2班 | [3] | [2] | [3] | 键盘有灰尘 |
| ... | ... | ... | ... | ... |

**优势**：
- 🚀 可以快速浏览整个年级的情况
- 🚀 一次性完成所有班级评分
- 🚀 便于横向对比不同班级

### 第三步：填写评分备注

为每个班级填写具体的备注说明：

**毕业班评分重点**：

对于毕业班（初三、高三、高三VCE），应特别关注：

✅ **信息素养**：
- 是否养成良好的电脑使用习惯
- 对设备的爱护程度
- 规范使用意识

✅ **规范使用**：
- 是否遵守机房管理制度
- 椅子归位、关闭电源等细节
- 离开机房时的整体状态

**备注示例**：

对于毕业班：
- "整体信息素养良好，电脑保护意识强"
- "建议加强椅子归位管理，其他方面优秀"
- "毕业班级榜样，建议继续保持"

对于非毕业班：
- "键盘有轻微灰尘，建议每日清洁"
- "物品摆放整齐，使用规范，继续保持"
- "垃圾桶已满，建议及时清理"

### 第四步：提交评分

1. 确认所有班级的分数已填写
2. 检查备注是否详细准确
3. 点击页面底部的 **"提交评分"** 按钮
4. 看到成功提示，评分完成

**提交后的特殊说明**：
- ✨ 评分自动标记为"新媒体委员"来源
- ✨ 权重自动设置为1.5倍
- ✨ 可以在"我的评分"中查看历史
- ⚠️ 同一周期内可重新提交，会覆盖之前的评分

## 🎓 毕业班专项检查

### 为什么要专项检查毕业班？

毕业班级即将离校，专项检查的目的：
1. 评估毕业生的信息素养水平
2. 了解多年培养的成效
3. 为毕业评价提供参考
4. 树立榜样，激励其他年级

### 毕业班评分要点

**信息素养考察**：
- 是否养成良好的电脑使用习惯？
- 对设备的爱护和保护意识如何？
- 是否能主动维护机房环境？
- 规范使用意识是否深入人心？

**具体检查项**：

✅ **设备保护**：
- 键盘、鼠标、显示屏保护得当
- 无人为损坏痕迹
- 线材整理规范

✅ **环境维护**：
- 主动保持桌面整洁
- 使用后及时清理
- 椅子归位、关闭电源

✅ **规范意识**：
- 遵守机房使用规定
- 不随意移动设备
- 正确使用各类设施

### 评分建议

**对于表现优秀的毕业班**：
- 在备注中给予表扬和肯定
- 可以作为其他年级的榜样
- 建议学校进行表彰

**对于需要改进的方面**：
- 具体指出问题所在
- 提出改进建议
- 给予改正的机会

## ⏰ 评分周期说明

### 周期机制

新媒体委员的评分周期与信息委员相同：

**周期类型**：
- 双周期：14天一个周期
- 单周期：7天一个周期

**周期规则**：
- 同一周期内可多次评分同一班级
- 新评分会覆盖该周期的旧评分
- 被覆盖的评分保存到历史记录

### 评分策略

**建议的评分频率**：

**常规评分**：
- 每个周期可以选择1-2个年级进行重点检查
- 不必每次都评所有年级

**专项检查**：
- 毕业班级建议每周期至少评一次
- 发现问题的年级可以增加评分频率

**时机选择**：
- 可以选择在信息委员评分后进行
- 也可以独立安排评分时间

## 🔍 常见问题

### Q1: 新媒体委员的1.5倍权重是如何计算的？

**A:** 
- 假设某班级收到：信息委员评分8分，新媒体委员评分9分
- 计算方式：(8×1.0 + 9×1.5) / (1.0+1.5) = (8+13.5) / 2.5 = 8.6分
- 新媒体委员的评分对最终成绩影响更大

### Q2: 我可以只评部分年级吗？

**A:** 可以。新媒体委员不需要每次评所有年级，可以根据需要选择重点年级进行评分。建议：
- 毕业班级每周期至少评一次
- 其他年级可轮换评分
- 发现问题的年级增加评分频率

### Q3: 同一个班级可以在一个周期内评多次吗？

**A:** 可以。与信息委员相同，新评分会覆盖该周期内的旧评分，被覆盖的评分会保存到历史记录中。

### Q4: 我的评分会不会与信息委员的评分冲突？

**A:** 不会冲突。系统会同时保留信息委员和新媒体委员的评分，然后按权重计算加权平均值作为最终成绩。

### Q5: 为什么要对毕业班进行专项评估？

**A:** 
- 毕业班即将离校，评估他们的信息素养水平
- 了解多年信息教育的培养成效
- 为毕业评价提供参考依据
- 树立榜样，激励其他年级学生

### Q6: 评分时如何体现"专项检查"的特点？

**A:** 
- 关注点更深入（不只是表面整洁）
- 从信息素养角度评估
- 重点关注规范使用和习惯养成
- 提出更专业的改进建议

### Q7: 如果发现信息委员评分有明显问题怎么办？

**A:** 
- 可以对该班级重新评分
- 在备注中说明情况
- 建议联系管理员核实
- 必要时可以进行复查

# 干事使用手册

本手册面向干事（信息委员）用户，介绍如何使用系统进行电脑卫生评分。

---

## 📋 干事职责

干事（信息委员）在系统中的主要职责：
- 定期对班级电脑进行卫生评分
- 记录评分备注（发现的问题）
- 查看个人的评分历史
- 了解当前的评分周期和要求

---

## 🚀 快速开始

### 首次登录

1. 访问系统地址：`http://your-domain.com`
2. 使用干事账户登录：
   - 用户名：如 `g6c1`（六年级1班干事）
   - 密码：初始密码（首次登录后请修改）

3. 进入评分首页

**⚠️ 重要：首次登录后，请修改初始密码！**

### 界面概览

干事评分界面包含：
- **周期信息卡片**：当前周期号、类型、时间范围
- **待评分班级列表**：显示本周期需要评分的班级
- **评分表单**：输入分数和备注
- **历史记录**：查看之前的评分记录

---

## 📝 评分流程

### 查看待评分班级

1. 登录后进入首页
2. 在 **"待评分班级"** 区域可以看到：
   - 需要评分的班级列表
   - 每个班级的评分状态
   - 上一次评分的时间

### 开始评分

1. 点击要评分的班级卡片
2. 进入该班级的评分表单
3. 查看评分项目和标准

### 填写评分内容

**电脑卫生评分项目：**

1. **电脑整洁度** （满分30分）
   - 键盘、鼠标、显示屏是否干净
   - 没有灰尘、污渍
   - 接线整齐

2. **物品摆放** （满分30分）
   - 电脑周围物品摆放是否整齐
   - 没有杂物堆放
   - 垃圾及时处理

3. **使用规范** （满分30分）
   - 是否有不规范使用现象
   - 椅子摆放是否整齐
   - 有无损坏设备

4. **其他** （满分10分）
   - 其他需要注意的问题
   - 特殊情况或改进

**评分标准：**
- 优秀（90-100分）：各项都做得很好
- 良好（75-89分）：总体较好，有个别小问题
- 一般（60-74分）：有较多问题需要改进
- 需改进（60分以下）：问题较多，需要立即改进

### 添加备注

1. 在 **"评分备注"** 文本框中记录发现的问题
2. 具体描述问题位置和内容
3. 例如：
   - "键盘有灰尘，显示屏上有污渍"
   - "椅子未摆整齐，地面有纸屑"
   - "网线连接松动，需要重新整理"

### 提交评分

1. 确认分数填写完毕
2. 查看备注是否清楚准确
3. 点击 **"提交评分"** 按钮
4. 系统提示提交成功

**提交后：**
- 不能再修改该班级本周期的评分
- 下一个周期可以重新评分
- 同一周期内的重复评分会覆盖之前的记录

## 📊 查看评分记录

### 个人评分历史

1. 点击顶部导航栏的 **"我的评分"**
2. 查看自己提交的所有评分记录
3. 显示内容包括：
   - 班级名称
   - 评分周期
   - 评分日期
   - 评分分数
   - 评分备注

### 搜索历史记录

1. 使用 **"班级筛选"** 找特定班级的记录
2. 使用 **"周期筛选"** 找特定周期的记录
3. 使用 **"日期筛选"** 找特定时间段的记录

### 查看详细信息

点击某条评分记录，可以查看：
- 详细的评分项目和分数
- 备注说明
- 提交时间
- 是否被覆盖（同周期新评分覆盖）

## ⏰ 评分周期说明

### 什么是评分周期？

- **周期**：系统规定的评分时间段
- **双周期**：14天为一个周期
- **单周期**：7天为一个周期
- **周期号**：用数字标识，从0开始

### 查看当前周期

在评分首页的 **"当前周期"** 卡片可以看到：
- 周期号：如第3周期
- 周期类型：单周或双周
- 周期时间范围：开始日期 - 结束日期
- 周期剩余天数

### 周期与评分的关系

- **同一周期内**：只能提交一次评分
- **重复评分**：同周期内新的评分会覆盖旧的
- **历史保留**：被覆盖的评分会保存到历史记录
- **新周期开始**：可以为同一班级重新评分

**示例：**
```
第2周期（1月1日-1月14日）
├─ 1月5日：提交分数80分
├─ 1月8日：更新分数为85分（覆盖之前的80分）
└─ 1月12日：最终分数85分被记录

第3周期（1月15日-1月28日）
└─ 可以重新评分
```

## 🔐 账户安全

### 修改密码

1. 点击右上角用户名
2. 选择 **"修改密码"**
3. 输入当前密码和新密码
4. 点击 **"确认修改"**

### 安全建议

- 使用强密码（至少8位，包含字母和数字）
- 定期更改密码（建议每学期一次）
- 不要与他人共享账户
- 使用完毕后退出登录

### 遗忘密码

如果遗忘密码：
1. 点击登录页面的 **"忘记密码"**
2. 输入用户名
3. 联系管理员重置密码
4. 或使用初始密码登录

## 🎯 评分要点

### 评分标准

**优秀（90-100分）：**
- ✅ 所有电脑都很干净
- ✅ 物品摆放整齐
- ✅ 没有发现问题

**良好（75-89分）：**
- ✅ 基本卫生状况良好
- ⚠️ 个别电脑有轻微灰尘
- ⚠️ 有个别物品摆放不齐

**一般（60-74分）：**
- ⚠️ 多个电脑需要清洁
- ⚠️ 物品摆放较乱
- ⚠️ 有使用规范问题

**需改进（60分以下）：**
- ❌ 卫生状况较差
- ❌ 问题较多，需要立即改进
- ❌ 可能有设备损坏

### 评分建议

1. **全面检查**：逐个检查班级的每台电脑
2. **认真记录**：在备注中详细说明问题
3. **公平公正**：按相同标准评分
4. **及时反馈**：及时向班级反馈评分结果
5. **表扬激励**：表扬卫生状况好的班级

## 💡 使用技巧

### 提高评分效率

1. **制定计划**：计划何时评分哪些班级
2. **提前准备**：了解班级位置和班级名称
3. **批量评分**：在一次巡查中评分多个班级
4. **及时提交**：完成评分后立即提交

### 准确记录问题

1. **具体描述**：不要写"脏"，要写"键盘有灰尘"
2. **标注位置**：说明具体位置"第2排左边第3台"
3. **建议改进**：在备注中提出具体改进建议
4. **跟踪改进**：下一周期查看是否改进

### 与班级沟通

1. **解释评分**：向班级解释为什么这样评分
2. **表扬优点**：肯定班级做得好的方面
3. **指出问题**：直言班级存在的问题
4. **给出建议**：提出具体的改进建议

## 🔍 常见问题

### Q1: 同一班级一个周期能评分多次吗？

**A:** 能。但新的评分会覆盖之前的评分。被覆盖的评分会保存到历史记录中。

### Q2: 如果评分后发现有误怎么办？

**A:** 在同一周期内可以重新提交评分，新评分会覆盖旧评分。如果是已经结束的周期，请联系管理员。

### Q3: 为什么登录后看不到某个班级？

**A:** 可能是：
- 该班级不在本周期的评分范围内
- 系统中没有配置该班级
- 您没有该班级的评分权限
- 请联系班主任或管理员确认

### Q4: 评分分数是如何计算的？

**A:** 总分 = 电脑整洁度 + 物品摆放 + 使用规范 + 其他 = 满分100分

### Q5: 可以查看其他干事的评分吗？

**A:** 不能。您只能查看自己的评分记录。如需查看其他信息，请联系班主任或管理员。

### Q6: 忘记提交评分怎么办？

**A:** 
- 在周期内任何时间都可以提交
- 周期结束后无法再提交该周期的评分
- 请在周期结束前及时提交

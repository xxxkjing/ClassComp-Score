# 教师使用手册

本手册面向教师用户，介绍如何使用教师监控面板查看和分析评分数据。

---

## 📋 教师职责

教师在系统中的主要职责：
- 监控所管理年级的评分进度
- 查看各班级评分统计数据
- 分析评分趋势和问题
- 导出班级评分报告
- 督促干事完成评分任务

---

## 🚀 快速开始

### 首次登录

1. 访问系统地址：`http://your-domain.com`
2. 使用教师账户登录：
   - 用户名：如 `t6`（六年级教师）
   - 密码：初始密码（首次登录后请修改）

3. 进入教师监控面板

**⚠️ 重要：首次登录后，请修改初始密码！**

### 界面概览

教师监控面板包含：
- **统计卡片**：今日评分、本周期完成率、年级平均分
- **趋势图表**：近期评分趋势、班级对比
- **数据表格**：详细评分记录
- **筛选工具**：按周期、班级筛选数据

---

## 📊 查看评分数据

### 今日评分概览

在页面顶部的统计卡片中可以看到：
- **今日评分数**：今天提交的评分总数
- **本周期完成率**：当前周期评分完成百分比
- **年级平均分**：所管理年级的平均得分

### 评分进度查看

1. 查看 **"评分完成情况"** 表格
2. 了解各班级的评分状态：
   - ✅ 已完成：显示绿色标记和分数
   - ⏳ 待评分：显示黄色标记
   - ❌ 超期未评：显示红色警告

### 按周期查看

1. 使用页面上方的 **"周期选择器"**
2. 选择要查看的周期：
   - 当前周期
   - 历史周期（按周期号选择）
3. 系统会显示该周期的所有评分数据

---

## 📈 数据统计分析

### 趋势图表

**评分趋势图**
- 显示近7天或近30天的评分数量变化
- 可以看出评分活跃度的变化趋势
- 帮助识别评分高峰和低谷期

**班级对比图**
- 横向对比各班级的评分情况
- 柱状图显示各班级的平均分
- 快速识别表现优秀和需要改进的班级

### 详细数据表格

在 **"评分记录"** 表格中可以看到：
- 班级名称
- 评分时间
- 评分干事
- 评分分数
- 评分备注

**表格功能：**
- **排序**：点击列标题按升序/降序排序
- **搜索**：使用搜索框查找特定班级或干事
- **分页**：浏览大量数据记录

---

## 🔍 数据筛选

### 按班级筛选

1. 点击 **"班级筛选"** 下拉菜单
2. 选择要查看的班级
3. 系统显示该班级的所有评分记录

### 按时间筛选

1. 选择时间范围：
   - 今日
   - 本周
   - 本周期
   - 自定义日期范围
2. 查看筛选后的数据

### 组合筛选

可以同时使用多个筛选条件：
- 周期 + 班级
- 时间范围 + 班级
- 获得更精确的数据分析

---

## 📥 导出报告

### 导出 Excel 报告

1. 点击页面上的 **"导出报告"** 按钮
2. 选择导出范围：
   - 当前周期数据
   - 所有历史数据
   - 自定义周期范围
3. 点击 **"导出"**

**报告内容：**
- 评分详情：所有评分记录
- 班级汇总：各班级平均分、最高分、最低分
- 趋势分析：评分变化趋势

### 报告用途

- **周例会**：展示本周评分情况
- **家长会**：说明班级管理成效
- **学期总结**：分析整体趋势
- **问题诊断**：识别需要关注的班级

## 🎯 监控要点

### 日常监控

**每日检查：**
- [ ] 查看今日评分数量
- [ ] 确认是否有未评分班级
- [ ] 查看是否有异常低分

**每周检查：**
- [ ] 查看本周期完成率
- [ ] 分析评分趋势
- [ ] 关注持续低分的班级

### 问题识别

**需要关注的情况：**
1. **评分停滞**：某班级长期未评分
2. **分数异常**：突然的高分或低分
3. **评分频率异常**：过于频繁或稀少
4. **备注警告**：评分备注中的严重问题

### 干预措施

**发现问题后：**
1. 联系相关干事了解情况
2. 实地检查问题班级
3. 与班主任沟通
4. 必要时请管理员协助

## 🔐 账户安全

### 修改密码

1. 点击右上角用户名
2. 选择 **"修改密码"**
3. 输入当前密码和新密码
4. 点击 **"确认修改"**

### 安全建议

- 使用强密码（至少8位，包含字母和数字）
- 定期更改密码（建议每学期一次）
- 不要与他人共享账户
- 使用完毕后退出登录

## 🔍 常见问题

### Q1: 为什么看不到其他年级的数据？

**A:** 教师只能查看自己所管理年级的数据。如需查看其他年级，请联系管理员调整权限。

### Q2: 评分完成率是如何计算的？

**A:** 完成率 = 已评分班级数 / 应评分班级总数 × 100%

### Q3: 如何理解评分趋势图？

**A:** 
- 上升趋势：评分活跃度增加
- 下降趋势：评分活跃度减少
- 波动：可能与周末、节假日有关

### Q4: 导出的报告包含哪些数据？

**A:** 包含选定时间范围内的所有评分记录、统计数据和趋势分析。

### Q5: 可以修改评分记录吗？

**A:** 教师没有修改评分的权限。如需修改，请联系管理员。

### Q6: 如何督促干事完成评分？

**A:** 
1. 通过监控面板识别未完成评分的班级
2. 联系对应的干事
3. 了解未评分原因
4. 必要时向管理员反馈

## 💡 使用技巧

### 高效监控

1. **设置每日查看时间**：如每天下午固定时间检查
2. **关注完成率**：重点关注完成率低于80%的周期
3. **定期导出报告**：每周或每月导出一次作为档案
4. **善用筛选功能**：快速定位特定班级或时间段的数据

### 数据分析

1. **对比分析**：比较不同班级的评分趋势
2. **时间分析**：观察评分在不同时间段的变化
3. **问题诊断**：通过数据识别管理问题
4. **效果评估**：评估管理措施的效果

### 沟通协作

1. **与干事沟通**：及时反馈评分情况
2. **与班主任协调**：共同解决班级问题
3. **与管理员配合**：报告系统问题或建议改进

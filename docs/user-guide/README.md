## 👥 角色导航

ClassComp Score 系统支持三种用户角色，每种角色都有专属的使用手册：

### 👨‍🎓 信息委员（Info Commissioner）
**基础评分人员 | 权重：1.0x**

按照评分链条对指定年级班级进行评分，是系统的主要评分力量。

📖 **[信息委员使用手册](info-commissioner-guide.md)**

**主要功能：**
- ✅ 按评分链条评分（中预→初一→初二→中预，高一↔高二，高一VCE↔高二VCE）
- ✅ 查看个人评分历史
- ✅ 在评分周期内覆盖之前的评分
- ✅ 移动端随时评分

---

### 📹 新媒体委员（New Media Officer）
**特殊评分人员 | 权重：1.5x**

可以对所有年级进行评分，评分权重更高，适合专项检查。

📖 **[新媒体委员使用手册](new-media-officer-guide.md)**

**主要功能：**
- ✅ 评所有年级（不受评分链条限制）
- ✅ 1.5倍评分权重
- ✅ 评分来源标记为"新媒体委员"
- ✅ 毕业班级专项检查
- ✅ 移动端随时评分

---

### 👨‍💼 管理员（Administrator）
**系统管理员 | 最高权限**

负责系统配置、用户管理和数据维护，是系统的运维管理者。

📖 **[管理员使用手册](admin-guide.md)**

**主要功能：**
- ✅ 用户管理（创建、删除、重置密码）
- ✅ 学期配置（学期信息、评分周期、参与班级）
- ✅ 权重配置（新媒体委员和信息委员的评分权重）
- ✅ 数据导出和备份
- ✅ 查看所有统计数据
- ✅ 周期类型动态切换

## 📖 评分流程概览

### 信息委员评分流程

```
1. 登录系统
   ↓
2. 查看待评分班级（按评分链条显示）
   ↓
3. 选择班级，填写评分表单
   - 电脑整洁度（0-3分）
   - 物品摆放（0-3分）
   - 使用规范（0-4分）
   - 备注说明
   ↓
4. 提交评分
   ↓
5. 查看评分历史
```

### 新媒体委员评分流程

```
1. 登录系统
   ↓
2. 选择要评分的年级
   ↓
3. 为该年级所有班级批量评分
   - 电脑整洁度（0-3分）
   - 物品摆放（0-3分）
   - 使用规范（0-4分）
   - 备注说明
   ↓
4. 提交评分（权重1.5x）
   ↓
5. 查看评分历史
```

## 💡 使用建议

### 对信息委员
- 📅 了解当前评分周期，在周期内及时完成评分
- 📝 详细填写评分备注，帮助班级改进
- 📱 利用移动端随时随地评分
- 🔄 发现错误时可在同周期内重新提交覆盖

### 对新媒体委员
- 🎯 利用高权重优势进行重点检查
- 🎓 对毕业班级进行专项评估
- 📊 关注评分数据，发现管理问题
- 📋 评分时明确标注具体问题

### 对管理员
- ⚙️ 学期开始前完成所有配置
- 💾 定期备份数据（建议每周一次）
- 🔐 定期检查用户账户安全
- 📈 关注评分完成率，及时提醒

## 🔍 常见问题

### Q1: 忘记密码怎么办？
**A:** 联系管理员使用密码重置功能。

### Q2: 评分提交后可以修改吗？
**A:** 在同一评分周期内可以重新提交，新评分会覆盖旧评分。历史记录会被保留。

### Q3: 为什么我看不到某些班级？
**A:** 
- 信息委员：只能看到评分链条中应该评分的班级
- 新媒体委员：可以看到所有年级，按年级选择评分
- 可能该班级不在当前学期的参与班级配置中

### Q4: 评分周期是什么意思？
**A:** 评分周期是管理员设置的评分时间段（单周7天或双周14天）。同一周期内对同一班级的多次评分会互相覆盖，以最后一次为准。

### Q5: 新媒体委员的1.5倍权重如何计算？
**A:** 最终成绩 = (信息委员评分总和 × 1.0 + 新媒体委员评分总和 × 1.5) / 评分总次数的权重和
